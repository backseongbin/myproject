# 🤖 복지로봇 - 정부지원금 자동 매칭 서비스

나이, 거주지, 소득, 가구 형태 등 개인 정보를 입력하면 받을 수 있는 국가 및 지자체 지원금을 자동으로 찾아주는 웹 서비스입니다.

## 화면 미리보기

| 입력 화면 | 결과 화면 |
|---|---|
| 개인 정보 4단계 입력 | 매칭된 지원금 카드 목록 |

## 주요 기능

- **자동 매칭** – 입력한 조건(나이·소득·가구·지역·취업상태 등)을 기반으로 수령 가능한 지원금을 즉시 검색
- **매칭 점수** – 각 지원금과 내 조건의 일치도를 퍼센트로 표시
- **필터 & 정렬** – 국가 / 지자체 / 청년 / 노인 / 가족 카테고리별 필터, 우선순위·금액·카테고리 정렬
- **신청 바로가기** – 각 지원금의 공식 신청 페이지로 직접 연결
- **개인정보 보호** – 모든 처리가 브라우저 내에서만 이루어지며 외부 서버로 데이터를 전송하지 않음

## 지원 혜택 목록 (2025년 기준)

| 분류 | 주요 지원금 |
|---|---|
| 국가 기초 | 생계급여, 주거급여, 의료급여 |
| 청년 | 국민취업지원제도, 청년도약계좌, 청년 월세 지원 |
| 가족·육아 | 아동수당, 영아수당, 첫만남이용권, 임신바우처 |
| 노인·장애 | 기초연금, 장애인연금, 노인 장기요양 |
| 한부모 | 한부모가족 양육비 지원 |
| 지자체 | 서울 임산부 교통비, 서울 안심소득, 경기 청년 복지포인트, 부산 청년 취업장려금 등 |

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라, 프레임워크 없음)
- 외부 의존성 없음 – 서버 불필요, 파일만 열어도 동작

## 실행 방법

### 방법 1. 파일 직접 열기

```
index.html 파일을 브라우저로 열기
```

### 방법 2. 로컬 서버 실행 (권장)

```bash
# Node.js가 있는 경우
npx serve . -l 3000
# 브라우저에서 http://localhost:3000 접속
```

## 파일 구조

```
├── index.html   # 전체 UI 마크업
├── style.css    # 스타일시트
├── app.js       # 매칭 엔진 및 UI 로직
└── data.js      # 지원금 데이터베이스 및 지역 데이터
```

## 매칭 알고리즘

각 지원금 항목에 대해 아래 조건을 순서대로 검사하고, 모든 조건을 통과하면 점수를 계산합니다.

1. 나이 범위 (minAge / maxAge)
2. 소득 분위 (maxIncomeLevel)
3. 거주 지역
4. 가구 유형
5. 취업 상태
6. 주거 형태
7. 추가 사항 (임산부·장애인·출산 등)

지역·소득·추가 사항이 정확히 일치할수록 매칭 점수가 올라갑니다.
